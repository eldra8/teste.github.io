Bonjour et bienvenu sur ce site entièrement d'édié à BloodGame !!!

C'est très simple.Si vous voulez réserver un escape game,alors cliquez sur le boutton "réserver" puis sélectionner l'escape game à réserver.

Pour voir l'escape game disponible en ce moment même,cliquez sur le boutton "en ce moment même" puis réserver(moin de temps d'attente).

Pour demmender de crée un escape game,cliquez sur le boutton "création".

Vous pouvez allez voir le nombre de point que vous avez remportez dans "mes point" et les dépenser dans "dépenser".

Bonne chance !
